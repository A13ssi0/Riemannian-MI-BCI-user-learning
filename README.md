# Riemannian-MI-BCI-user-learning
Repository for the manuscript **“The Riemannian Geometry of User Learning in MI-BCI: a Cybathlon Longitudinal Study.”**  This repository contains the longitudinal analysis of MI-BCI using Riemannian EEG covariance geometry, geodesic distance, tangent-space angular metrics, diffusion embeddings, clustering, and spatial EEG structure interpretation.
